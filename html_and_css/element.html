HTML 块

HTML块被定义为：块元素 或 内联元素

  • HTML块
    • 块元素
      • 总是独占一行，表现为另起一行开始，而且其后的元素也必须另起一行；
      • 宽度（width）,高度（height）,内边距（padding）,外边距（margin)都可控
      • etc:h1 p ul table 标签
    • 内联元素
      • 和相邻的内联元素同一行
      • 宽度（width）,高度（height）,内边距（padding）,外边距（margin)不可改变
      • etc: b td a img 标签
